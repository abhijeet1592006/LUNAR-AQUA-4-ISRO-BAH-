# 1. Core Lunar Datasets

## Chandrayaan-2 (ISRO)

Official data portal

https://pradan.issdc.gov.in/ch2/

### Terrain Mapping Camera-2 (TMC-2)

Purpose
- Stereo terrain mapping
- DEM/DTM generation
- Crater detection
- Slope analysis

Useful for

- Terrain reconstruction
- Shadow simulation
- Landing site analysis

---

### DFSAR

Purpose

Dual-frequency Synthetic Aperture Radar

Useful for

- Subsurface investigation
- Ice probability estimation
- Radar backscatter analysis

---

### IIRS

Purpose

Imaging Infrared Spectrometer

Useful for

- Water/OH signatures
- Mineral identification
- Spectral analysis

---

### OHRC

Purpose

Very high-resolution optical imagery

Useful for

- Validation
- Small crater inspection
- Surface texture analysis

---

## NASA Lunar Reconnaissance Orbiter (LRO)

Official Portal

https://pds.nasa.gov/

### LOLA

Purpose

Global lunar elevation model

Useful for

- Polar topography
- DEM comparison
- Illumination studies

Documentation

https://pds-geosciences.wustl.edu/missions/lro/lola.htm

---

### Diviner

Purpose

Surface temperature measurements

Useful for

- Cold traps
- Thermal analysis

---

### Mini-RF

Purpose

Radar observations

Useful for

- CPR analysis
- Ice studies

---

### LROC

Official Site

https://www.lroc.asu.edu/

Useful for

- High-resolution imagery
- Landing site validation
